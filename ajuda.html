Funcionalidades Principais

  • Cálculo de Volume

    Calcule o volume de madeira serrada com precisão usando as dimensões fornecidas. Suporte para diferentes unidades e tipos de madeira.

  • Lista de Itens

    Organize seus cálculos em listas detalhadas para melhor controle e acompanhamento de projetos.

  • Geração de PDF

    Exporte seus orçamentos e listas em formato PDF profissional com logo personalizado.

  • Compartilhamento

    Compartilhe suas listas e orçamentos com clientes e colaboradores de forma rápida e segura.

  • Relatórios Detalhados

    Gere relatórios completos com informações sobre seus cálculos, custos e estatísticas de uso.

  • Backup Automático

    Seus dados são automaticamente salvos e sincronizados para maior segurança.

Perguntas Frequentes

  • Como usar a calculadora de madeira?

    Para usar a calculadora, siga estes passos:

    1. Digite as dimensões da madeira (espessura, largura e comprimento) em centímetros
    2. Informe a quantidade de peças desejada
    3. Digite o preço por metro cúbico (m³)
    4. Clique em "Calcular" para ver o resultado automaticamente
    5. Use "Adicionar à Lista" para compilar múltiplos itens
  • Como salvar e gerenciar orçamentos?

    Para gerenciar seus orçamentos:

    1. Preencha as informações do cliente (nome, contato, condições)
    2. Adicione todos os itens de madeira à lista
    3. Clique em "Salvar Orçamento"
    4. Acesse a seção "Orçamentos" para visualizar salvos
    5. Use as opções de editar, duplicar ou gerar PDF

    Todos os orçamentos ficam salvos localmente e podem ser acessados a qualquer momento.

  • Como personalizar meu perfil e empresa?

    Para personalizar seu perfil:

    1. Acesse a seção "Perfil" no menu lateral
    2. Preencha suas informações pessoais e da empresa
    3. Faça upload do logo da sua empresa
    4. Configure informações de contato e endereço
    5. Clique em "Salvar Perfil"

    Estas informações serão utilizadas nos PDFs gerados automaticamente.

  • Quais são as diferenças entre os planos?

    Oferecemos três planos para atender suas necessidades:

    • Gratuito: Funcionalidades básicas, até 10 orçamentos salvos
    • Premium: Orçamentos ilimitados, PDF com logo, backup na nuvem
    • Empresarial: Todas as funcionalidades + suporte prioritário e recursos avançados

    Acesse a seção "Planos" para ver todos os detalhes e fazer upgrade.

  • Como funciona o sistema offline?

    A Calculadora de Madeira funciona como uma Progressive Web App (PWA):

    • Todos os dados são salvos localmente no seu dispositivo
    • Funciona mesmo sem conexão com a internet
    • Pode ser instalada como um app no seu celular ou computador
    • Sincronização automática quando voltar online (planos pagos)
  • Como exportar dados e fazer backup?

    Para proteger seus dados:

    1. Acesse "Configurações" no menu
    2. Use "Exportar Configurações" para baixar um backup
    3. Planos pagos têm backup automático na nuvem
    4. PDFs podem ser salvos individualmente
    5. Use "Importar Configurações" para restaurar dados
