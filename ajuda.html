Perguntas Frequentes

  • Como usar a calculadora?
    Para usar a calculadora, siga estes passos:
    1. Digite as dimensões da madeira (largura, espessura e comprimento)
    2. Informe a quantidade de peças
    3. Digite o preço por m³
    4. Clique em "Calcular" para ver o resultado
  • Como salvar uma lista de cálculos?
    Para salvar uma lista de cálculos:
    1. Realize os cálculos desejados
    2. Clique em "Salvar Lista"
    3. Digite um nome para a lista
    4. Clique em "Confirmar"
    A lista será salva e poderá ser acessada posteriormente na seção "Minhas Listas".
  • Como alterar minhas informações pessoais?
    Para alterar suas informações pessoais:
    1. Acesse a seção "Perfil"
    2. Clique em "Editar Perfil"
    3. Altere as informações desejadas
    4. Clique em "Salvar Alterações"
  • Como funciona o plano premium?
    O plano premium oferece recursos adicionais:
    • Listas ilimitadas de cálculos
    • Exportação de relatórios em PDF
    • Backup automático na nuvem
    • Suporte prioritário
    Para assinar o plano premium, acesse a seção "Planos" e escolha o plano desejado.
